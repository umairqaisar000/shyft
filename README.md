# Shyft – Company Portfolio

A single-page dark-theme portfolio site for Shyft, built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- **Dark theme** – Consistent dark palette with accent highlights
- **Navbar** – Fixed header with smooth scroll links and mobile menu
- **Hero** – Headline and CTA on the left with full-bleed background image
- **How we work** – Three-card layout (Discover, Design & Build, Launch & Scale)
- **Our services** – Four service cards with scroll-triggered animation
- **Our team** – Three team members with photo, name, role, and short bio
- **FAQs** – Expandable accordion
- **Book a call** – Contact form (name, email, message)
- **Footer** – Links and copyright
- **Responsive** – Layout and typography tuned for mobile, tablet, and desktop

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Tech stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Next/Image for optimized images (Unsplash)
